## 1. Create an API token

<Steps>
  <Step title="Open the dashboard">
    Sign in at [mex-pet-app.vercel.app](https://mex-pet-app.vercel.app) and navigate to **Settings → API Keys**.
  </Step>
  <Step title="Create a token">
    Click **Create token**, give it a descriptive name (e.g. `Zapier production`), and choose the [scopes](/api-tokens#scope-catalog) it needs. Grant only what the integration requires.
  </Step>
  <Step title="Copy the secret">
    The full token (`mxp_live_…`) is shown **once**. Copy it immediately and store it somewhere safe — you cannot view it again. If you lose it, revoke it and create a new one.
  </Step>
</Steps>

## 2. Make your first request

Send the token as a bearer token in the `Authorization` header.

<CodeGroup>

```bash cURL
curl https://mex-pet-app.vercel.app/api/v1/contacts \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```javascript Node.js
const res = await fetch(
  "https://mex-pet-app.vercel.app/api/v1/contacts",
  { headers: { Authorization: "Bearer mxp_live_your_token_here" } }
);
const { data } = await res.json();
console.log(data);
```

```python Python
import requests

res = requests.get(
    "https://mex-pet-app.vercel.app/api/v1/contacts",
    headers={"Authorization": "Bearer mxp_live_your_token_here"},
)
print(res.json()["data"])
```

</CodeGroup>

## 3. Create a contact

```bash cURL
curl -X POST https://mex-pet-app.vercel.app/api/v1/contacts \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "full_name": "John Smith", "email": "john@example.com", "phone": "+15125550199" }'
```

## 4. Send a message

```bash cURL
curl -X POST https://mex-pet-app.vercel.app/api/v1/messages \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "channel": "sms", "to": "+15125550199", "message": "Your pup is ready for pickup! 🐾" }'
```

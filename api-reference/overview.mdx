This is the complete reference for the **Mex Pet API**. Every endpoint has its own page with parameters, request body, response schema, and a **live playground** you can call directly with your token.

## Base URL

```text
https://mex-pet-app.vercel.app/api/v1
```

## Authentication

Every request needs your bearer token in the `Authorization` header:

```text
Authorization: Bearer mxp_live_…
```

Create one in **Settings → API Keys** and grant only the [scopes](/api-tokens) it needs. New to the API? Start with the [Quickstart](/quickstart).

## Using the playground

Open any endpoint page, paste your token into the **Authorization** field, fill in the parameters, and hit **Send** to run a real request against your ranch — no extra setup required.

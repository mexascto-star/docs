---
title: "Retrieve a note"
description: "Retrieves a single note by ID. Requires the notes.read scope."
openapi: "GET /notes/{id}"
---

Retrieves a single note by its ID.

**Required scope:** `notes.read`

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/notes/note_kk1f2g \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "note_kk1f2g",
  "object": "note",
  "contact_id": "usr_6f1c3d",
  "body": "Jane prefers afternoon pickups. Bella gets anxious during thunderstorms — extra attention needed.",
  "created_by": "stf_ee5f6a",
  "created_at": "2026-07-01T10:00:00.000Z"
}
```

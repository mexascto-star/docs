---
title: "List automations"
description: "Returns a paginated list of automation workflows. Requires the automations.read scope."
openapi: "GET /automations"
---

Returns all automation workflows in your ranch. Use this to audit which automations are active, paused, or in draft.

**Required scope:** `automations.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /automations
Authorization: Bearer mxp_live_your_token_here
```

## Status values

| Status | Meaning |
| --- | --- |
| `active` | Running — fires when trigger condition is met |
| `paused` | Temporarily stopped — trigger events are ignored |
| `draft` | Not yet activated |

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/automations \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

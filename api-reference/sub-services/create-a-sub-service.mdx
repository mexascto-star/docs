---
title: "Create a sub-service"
description: "Creates a sub-service (add-on / line item) under a parent service. Requires the services.write scope."
---

Creates a sub-service in the `sub_services` table and links it to a parent service. Sub-services represent add-ons, line items, or optional extras that belong to a service package.

**Required scope:** `services.write`

<Warning>
  `service_id` must be an existing service in your business. An invalid ID returns `400 invalid_request_error`.
</Warning>

<Tip>
  Send an `Idempotency-Key` header so retries don't create duplicates.
</Tip>

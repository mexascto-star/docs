---
title: "Create a sub-service"
description: "Creates a sub-service (add-on / line item) under a parent service. Requires the services.write scope."
---

Creates a sub-service in the `sub_services` table and links it to a parent service. Sub-services represent add-ons, line items, or optional extras that belong to a service package. The parent `service_id` must be an existing service in your business.

**Required scope:** `services.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

<Tip>
  Use an `Idempotency-Key` header to safely retry this request without creating duplicate sub-services.
</Tip>

## Request

```bash
POST /sub-services
Authorization: Bearer mxp_live_your_token_here
Content-Type: application/json
```

## Body fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `service_id` | string | ✅ | ID of the parent service. Must be an existing service in your business. |
| `name` | string | ✅ | Sub-service name |
| `description` | string | — | Full description |
| `price` | number | — | Price in dollars |
| `duration_minutes` | integer | — | Estimated duration |
| `sort_order` | integer | — | Display order (lower = first) |
| `is_active` | boolean | — | Whether available for selection (default `true`) |
| `pricing_model` | string | — | `fixed` (default) |
| `unit_label` | string | — | Label for the unit (e.g. `per dog`, `per hour`) |

<Warning>
  `service_id` must be an existing service in your business. An invalid ID returns `400 invalid_request_error`.
</Warning>

## Example

```bash
curl -X POST https://mex-pet-app.vercel.app/api/v1/sub-services \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -H "Idempotency-Key: a1b2c3d4-e5f6-7890-abcd-ef1234567890" \
  -d '{
    "service_id": "svc_gg7b8c",
    "name": "Bath & Brush",
    "description": "Full bath, blow dry, and brush-out.",
    "price": 39,
    "duration_minutes": 60,
    "sort_order": 1,
    "is_active": true
  }'
```

```json
  "object": "sub_service",
  "service_id": "svc_gg7b8c",
  "name": "Bath & Brush",

  "price": 39,
  "duration_minutes": 60,

  "is_active": true,
  "pricing_model": "fixed",
  "unit_label": null,
  "created_at": "2026-07-09T10:00:00.000Z"

```

## Error responses

| Status | Code | Meaning |
| --- | --- | --- |
| `400` | `invalid_request_error` | Missing required field or `service_id` not found in your business |
| `401` | `authentication_error` | Missing or invalid token |
| `403` | `permission_error` | Token missing `services.write` scope |

---
title: "List sub-services"
description: "Returns sub-services for a parent service. Requires the services.read scope."
---

Returns sub-services (add-ons and line items) from the `sub_services` table. Filter with `service_id` to get sub-services for a specific parent service. Sub-services are also embedded in `GET /services/{id}` responses.

**Required scope:** `services.read`

---
title: "List form submissions"
description: "Returns all submissions for a form. Requires the forms.read scope."
openapi: "GET /forms/{id}/submissions"
---

Returns all submissions for a specific form or survey. Each submission contains the raw answers and the `contact_id` of who submitted it.

**Required scope:** `forms.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /forms/{id}/submissions
Authorization: Bearer mxp_live_your_token_here
```

## Query parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/forms/frm_oo5j6k/submissions" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

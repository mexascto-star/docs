---
title: "List form submissions"
description: "Returns all submissions for a form. Requires the forms.read scope."
openapi: "GET /forms/{id}/submissions"
---

Returns all submissions for a specific form or survey. Each submission contains the raw answers keyed by field ID, plus the `contact_id` of the customer who submitted it (if known).

**Required scope:** `forms.read`

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` (path) | string | The form ID |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/forms/frm_oo5j6k/submissions?limit=5" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

---
title: "List forms & surveys"
description: "Returns a paginated list of forms and surveys. Requires the forms.read scope."
openapi: "GET /forms"
---

Returns all forms and surveys in your ranch. Use `?type=form` for intake forms and waivers, `?type=survey` for post-stay feedback.

**Required scope:** `forms.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /forms
Authorization: Bearer mxp_live_your_token_here
```

## Query parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |
| `type` | string | `form` or `survey` |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/forms?type=form" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

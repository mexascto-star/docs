---
title: "Delete a form"
description: "Permanently deletes a form or survey. Requires the forms.write scope."
openapi: "DELETE /forms/{id}"
---

Permanently removes a form or survey. Existing submissions are also deleted. Consider setting `status: "draft"` to unpublish a form without deleting it.

**Required scope:** `forms.write`

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/forms/frm_pp6k7l \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "frm_pp6k7l",
  "object": "form",
  "deleted": true
}
```

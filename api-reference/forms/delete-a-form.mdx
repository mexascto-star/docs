---
title: "Delete a form"
description: "Permanently deletes a form or survey. Requires the forms.write scope."
openapi: "DELETE /forms/{id}"
---

Permanently removes a form or survey. Consider setting `status: "draft"` to unpublish without deleting.

**Required scope:** `forms.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
DELETE /forms/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/forms/frm_pp6k7l \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

## Response `200`

```json
{
  "id": "frm_pp6k7l",
  "object": "form",
  "deleted": true
}
```

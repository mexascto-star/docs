title: "Create a form or survey"
description: "Creates a new form or survey. Requires the forms.write scope."
openapi: "POST /forms"
---

Creates a form or survey. Use `type: "form"` for intake forms and waivers, `type: "survey"` for post-stay feedback.

**Required scope:** `forms.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
POST /forms
Authorization: Bearer mxp_live_your_token_here
Content-Type: application/json
```

## Body fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | string | ✅ | Form title |
| `type` | string | — | `form` (default) or `survey` |
| `fields` | array | — | Field definitions |
| `settings` | object | — | Display and behavior settings |
| `status` | string | — | `draft` (default) or `published` |

## Example

```bash
curl -X POST https://mex-pet-app.vercel.app/api/v1/forms \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Post-Stay Feedback Survey",
    "type": "survey",
    "status": "published"
  }'
```

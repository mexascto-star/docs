---
title: "Retrieve a form"
description: "Retrieves a single form or survey by ID. Requires the forms.read scope."
openapi: "GET /forms/{id}"
---

Retrieves a form or survey by ID including its field definitions and settings.

**Required scope:** `forms.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /forms/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/forms/frm_oo5j6k \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

## Response `200`

```json
{
  "id": "frm_oo5j6k",
  "object": "form",
  "title": "New Pet Intake Form",
  "type": "form",
  "status": "published",
  "fields": [],
  "settings": {},
  "created_at": "2026-02-01T00:00:00.000Z"
}
```

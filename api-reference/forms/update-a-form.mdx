---
title: "Update a form"
description: "Updates a form or survey. Requires the forms.write scope."
openapi: "PATCH /forms/{id}"
---

Updates a form or survey. Use this to publish a draft, rename it, or update settings.

**Required scope:** `forms.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
PATCH /forms/{id}
Authorization: Bearer mxp_live_your_token_here
Content-Type: application/json
```

## Example — publish a draft

```bash
curl -X PATCH https://mex-pet-app.vercel.app/api/v1/forms/frm_pp6k7l \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "status": "published", "title": "Post-Stay Satisfaction Survey" }'
```

## Response `200`

```json
{
  "id": "frm_pp6k7l",
  "object": "form",
  "title": "Post-Stay Satisfaction Survey",
  "type": "survey",
  "status": "published",
  "fields": [],
  "settings": {},
  "created_at": "2026-07-09T22:00:00.000Z"
}
```

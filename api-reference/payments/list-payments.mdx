---
title: "List payments"
description: "Returns payments and invoices. Requires the payments.read scope."
openapi: "GET /payments"
---

Returns payments recorded in your ranch. Payments are **read-only** — they are created automatically when a booking is paid through the app.

**Required scope:** `payments.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /payments
Authorization: Bearer mxp_live_your_token_here
```

## Query parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |
| `contact_id` | string | Filter to a specific customer |
| `invoice_number` | string | Look up a specific invoice (e.g. `INV-20260709-00123`) |
| `status` | string | `paid`, `pending`, `refunded` |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/payments?contact_id=usr_6f1c3d&status=paid" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

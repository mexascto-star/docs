---
title: "Retrieve a payment"
description: "Retrieves a single payment by ID. Requires the payments.read scope."
openapi: "GET /payments/{id}"
---

Retrieves a payment by ID. Use this to confirm payment status or pull invoice details.

**Required scope:** `payments.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /payments/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/payments/pay_jj0e1f \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

  "status": "paid",
  "payment_date": "2026-07-09T18:00:00.000Z",

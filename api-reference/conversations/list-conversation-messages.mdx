---
title: "List messages in a conversation"
description: "Returns all messages in a conversation thread. Requires the conversations.read scope."
openapi: "GET /conversations/{id}/messages"
---

Returns all messages in a conversation thread in chronological order.

**Required scope:** `conversations.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /conversations/{id}/messages
Authorization: Bearer mxp_live_your_token_here
```

## Query parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/conversations/conv_rr8m9n/messages" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

---
title: "List messages in a conversation"
description: "Returns all messages in a conversation thread. Requires the conversations.read scope."
openapi: "GET /conversations/{id}/messages"
---

Returns all messages in a conversation thread in chronological order. Each message shows the sender, receiver, channel, and whether it has been read.

**Required scope:** `conversations.read`

## Message object fields

| Field | Description |
| --- | --- |
| `id` | Unique message ID |
| `conversation_id` | The parent conversation |
| `sender_id` | ID of the sender (staff or contact) |
| `receiver_id` | ID of the recipient |
| `channel` | `sms` or `email` |
| `subject` | Email subject line (email only, null for SMS) |
| `text` | Message body |
| `is_read` | Whether the message has been read |
| `created_at` | When the message was sent |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/conversations/conv_rr8m9n/messages" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

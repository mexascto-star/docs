---
title: "List conversations"
description: "Returns a paginated list of message threads with your contacts. Requires the conversations.read scope."
openapi: "GET /conversations"
---

Returns all conversation threads between your ranch and its contacts. Conversations are read-only — they are created automatically when a contact sends an inbound message or when you send one via `POST /messages`. Use `GET /conversations/{id}/messages` to fetch the full message history for a thread.

**Required scope:** `conversations.read`

## Conversation object fields

| Field | Description |
| --- | --- |
| `id` | Unique conversation ID |
| `customer_id` | The contact this thread belongs to |
| `staff_id` | Assigned staff member, if any |
| `channel` | `sms` or `email` |
| `last_message` | Preview of the most recent message |
| `last_message_time` | Timestamp of the most recent message |
| `created_at` | When the conversation was first created |

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/conversations \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

---
title: "Retrieve a staff member"
description: "Retrieves a single staff member by ID. Requires the staff.read scope."
openapi: "GET /staff/{id}"
---

Retrieves a staff member by ID. Use this to verify assignment details before updating a booking's `assigned_staff_id`.

**Required scope:** `staff.read`

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/staff/stf_ee5f6a \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "stf_ee5f6a",
  "object": "staff",
  "user_id": "auth_kk1f2a",
  "full_name": "Maria Lopez",
  "email": "maria@ranch.com",
  "phone": "+15125550300",
  "staff_type": "worker",
  "service_type": "boarding",
  "is_active": true,
  "is_online": true,
  "avatar_url": null,
  "created_at": "2026-03-01T08:00:00.000Z"
}
```

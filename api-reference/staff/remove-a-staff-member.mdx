---
title: "Remove a staff member"
description: "Removes a staff member from your ranch. Requires the staff.write scope."
openapi: "DELETE /staff/{id}"
---

Removes a team member. Consider using `PATCH` to set `is_active: false` instead — deactivating preserves assignment history on past bookings.

**Required scope:** `staff.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
DELETE /staff/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/staff/stf_ff6a7b \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

## Response `200`

```json
{
  "id": "stf_ff6a7b",
  "object": "staff",
  "deleted": true
}
```

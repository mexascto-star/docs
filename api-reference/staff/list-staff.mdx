---
title: "List staff"
description: "Returns your team members. Requires the staff.read scope."
openapi: "GET /staff"
---

Returns all team members in your ranch. Filter by `staff_type` to get workers, drivers, or receptionists separately, or use `active=true` to get only currently active members.

**Required scope:** `staff.read`

## Filters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |
| `staff_type` | string | `worker`, `driver`, `receptionist` |
| `active` | boolean | `true` = active members only |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/staff?staff_type=worker&active=true" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

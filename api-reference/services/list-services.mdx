---
title: "List services"
description: "Returns your service catalog. Requires the services.read scope."
openapi: "GET /services"
---

Returns your priced service catalog. Use `active=true` to get only the services currently available for booking. Filter by `category` to build category-specific views (e.g. show only grooming services in a booking form).

**Required scope:** `services.read`

## Filters

| Parameter | Type | Description |
| --- | --- | --- |
| `limit` | integer | Items to return, 1–100 (default 20) |
| `offset` | integer | Items to skip |
| `active` | boolean | `true` = active services only |
| `category` | string | `boarding`, `grooming`, `daycare`, `training`, `addon`, `retail` |
| `query` | string | Free-text search over service title |

## Example

```bash
curl "https://mex-pet-app.vercel.app/api/v1/services?active=true&category=boarding" \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

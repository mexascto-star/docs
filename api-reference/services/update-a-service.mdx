---
title: "Update a service"
description: "Updates a service in your catalog. Requires the services.write scope."
openapi: "PATCH /services/{id}"
---

Updates one or more fields on a service. Common use cases: toggling `is_active` to hide/show a service, adjusting pricing, or updating the description.

**Required scope:** `services.write`

## Example — deactivate a service

```bash
curl -X PATCH https://mex-pet-app.vercel.app/api/v1/services/svc_ff6a7b \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "is_active": false }'
```

## Example — update pricing

```bash
curl -X PATCH https://mex-pet-app.vercel.app/api/v1/services/svc_ff6a7b \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "base_price": 60,
    "description": "Overnight stay with 4 walks per day and bedtime treat."
  }'
```

---
title: "Delete a service"
description: "Permanently deletes a service. Requires the services.write scope."
openapi: "DELETE /services/{id}"
---

Permanently removes a service. Consider setting `is_active: false` instead to hide it without losing history.

**Required scope:** `services.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
DELETE /services/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/services/svc_gg7b8c \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

## Response `200`

```json
{
  "id": "svc_gg7b8c",
  "object": "service",
  "deleted": true
}
```

---
title: "Retrieve a service"
description: "Retrieves a single service by ID. Requires the services.read scope."
openapi: "GET /services/{id}"
---

Retrieves a service by ID. The response embeds a `sub_services` array — rows from the separate `sub_services` table where `service_id` matches this service's ID. For non-package services this array is empty `[]`.

**Required scope:** `services.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /services/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/services/svc_gg7b8c \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

---
title: "Create a service"
description: "Creates a new service in your catalog. Requires the services.write scope."
openapi: "POST /services"
---

Adds a new service to your catalog. Set `is_package: true` to mark it as a bundle. Set `parent_id` to make it a sub-service of an existing package.

**Required scope:** `services.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

<Note>
  The API writes the service name to **both** the `name` and `title` columns so it displays correctly in the app. Always supply `title` — it will appear as the service name everywhere in the Mex Pet dashboard.
</Note>

## Request

```bash
POST /services
Authorization: Bearer mxp_live_your_token_here
Content-Type: application/json
```

## Body fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | string | ✅ | Service name — written to both `name` and `title` columns so it shows in the app |
| `description` | string | — | Full description |
| `category` | string | — | `boarding`, `grooming`, `daycare`, `training`, `addon`, `retail` (saved lowercase) |
| `base_price` | number | — | Price in dollars |
| `price_type` | string | — | `nightly`, `fixed`, `tiered` |
| `price_unit` | string | — | `per_night`, `per_mile`, `fixed` |
| `is_active` | boolean | — | Available for booking (default `true`) |
| `is_package` | boolean | — | Whether this bundles sub-services |
| `parent_id` | string | — | ID of a parent service — makes this a **sub-service** of that package |

## Example — standalone service

```bash
curl -X POST https://mex-pet-app.vercel.app/api/v1/services \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Standard Boarding",
    "description": "Overnight stay in a private suite with 3 walks per day.",
    "category": "boarding",
    "base_price": 55,
    "price_type": "nightly",
    "price_unit": "per_night",
    "is_active": true
  }'
```

## Example — create a package then add sub-services

```bash
# 1. Create the package
curl -X POST https://mex-pet-app.vercel.app/api/v1/services \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Weekend Boarding Package",
    "category": "boarding",
    "base_price": 149,
    "price_type": "fixed",
    "price_unit": "fixed",
    "is_package": true
  }'

# 2. Add a sub-service using the package's id as parent_id
curl -X POST https://mex-pet-app.vercel.app/api/v1/services \
  -H "Authorization: Bearer mxp_live_your_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Bath & Brush",
    "category": "grooming",
    "base_price": 39,
    "price_type": "fixed",
    "price_unit": "fixed",
    "parent_id": "svc_gg7b8c"
  }'
```

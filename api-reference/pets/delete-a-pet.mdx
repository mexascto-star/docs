---
title: "Delete a pet"
description: "Permanently deletes a pet record. Requires the pets.write scope."
openapi: "DELETE /pets/{id}"
---

Permanently removes a pet record from your ranch. This action cannot be undone.

**Required scope:** `pets.write`

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/pets/pet_bb2c3d \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "pet_bb2c3d",
  "object": "pet",
  "deleted": true
}
```

A `404` is returned if the pet does not exist in your business.

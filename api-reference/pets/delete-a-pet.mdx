---
title: "Delete a pet"
description: "Permanently deletes a pet record. Requires the pets.write scope."
openapi: "DELETE /pets/{id}"
---

Permanently removes a pet record from your ranch.

**Required scope:** `pets.write`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
DELETE /pets/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/pets/pet_bb2c3d \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

## Response `200`

```json
{
  "id": "pet_bb2c3d",
  "object": "pet",
  "deleted": true
}
```

---
title: "Delete a booking"
description: "Permanently deletes a booking. Requires the bookings.write scope."
openapi: "DELETE /bookings/{id}"
---

Permanently removes a booking from your ranch. Consider updating `status` to `cancelled` instead if you need to keep a record.

**Required scope:** `bookings.write`

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/bookings/bkg_dd4e5f \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "bkg_dd4e5f",
  "object": "booking",
  "deleted": true
}
```

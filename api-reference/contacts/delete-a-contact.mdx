---
title: "Delete a contact"
description: "Permanently deletes a contact. Requires the contacts.write scope."
openapi: "DELETE /contacts/{id}"
---

Permanently removes a contact from your ranch. This action cannot be undone. Associated pets, bookings, and notes are not automatically deleted — remove them first if needed.

**Required scope:** `contacts.write`

<Warning>
  Deletion is permanent and immediate. There is no soft-delete or recovery.
</Warning>

## Example

```bash
curl -X DELETE https://mex-pet-app.vercel.app/api/v1/contacts/usr_9a2b4c \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

```json
{
  "id": "usr_9a2b4c",
  "object": "contact",
  "deleted": true
}
```

A `404` is returned if the contact does not exist in your business.

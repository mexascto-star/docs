---
title: "Retrieve a contact"
description: "Retrieves a single contact by ID. Requires the contacts.read scope."
openapi: "GET /contacts/{id}"
---

Retrieves a single contact by ID. The response includes a `pets` array — all pets registered under this contact — giving you a full customer profile in one request.

**Required scope:** `contacts.read`

**Base URL:** `https://mex-pet-app.vercel.app/api/v1`

## Request

```bash
GET /contacts/{id}
Authorization: Bearer mxp_live_your_token_here
```

## Path parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | string | The contact ID |

## Example

```bash
curl https://mex-pet-app.vercel.app/api/v1/contacts/usr_6f1c3d \
  -H "Authorization: Bearer mxp_live_your_token_here"
```

| Status | Code | Meaning |
| --- | --- | --- |
